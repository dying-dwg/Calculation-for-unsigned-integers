# Вычисление для целых беззнаковых чисел
### Постановка задачи
Произвести вычисления для беззнаковых целых чисел по формуле: $$\frac{A^2 + B^2}{C^3}$$
### Реализация
> Код написан на C с ассемблерной вставкой в программе VS 2019.
